class FractureTool

FractureTool class provides methods to fracture provided mesh and generate Blast asset data.

virtual bool setSourceMeshes(Mesh const *const *meshes, uint32_t meshesSize, const int32_t *ids = nullptr) = 0

Set input meshes which will be fractured, FractureTool will be reset.

If ids != nullptr, it must point to an array of length meshSizes. Each mesh will be assigned to a chunk with ID given by the corresponding element in ids. If the corresponding element is negative, or ids is NULL, then the chunk will be assigned an arbitrary (but currently unused) ID. Returns true iff all meshes were assigned chunks with valid IDs.

virtual void uniteChunks(uint32_t threshold, uint32_t targetClusterSize, const uint32_t *chunksToMerge, uint32_t mergeChunkCount, const NvcVec2i *adjChunks, uint32_t adjChunksSize, bool removeOriginalChunks = false) = 0

Optimize chunk hierarhy for better runtime performance.

It tries to unite chunks to groups of some size in order to transform flat hierarchy (all chunks are children of single root) to tree like hieracrhy with limited number of children for each chunk.

Parameters

  • threshold – [in] If number of children of some chunk less then maxAtLevel then it would be considered as already optimized and skipped.

  • targetClusterSize – [in] Target number of children for processed chunks.

  • chunksToMerge – [in] Which chunks are merge candidate. If NULL, all chunks will be a merge candidate.

  • mergeChunkCount – [in] size of chunksToMerge array, if chunksToMerge != NULL.

  • adjChunks – [in] Optional index pairs to describe chunk adjacency. May be NULL.

  • adjChunksSize – [in] If ‘adjChunks’ is not NULL, the number of index pairs in the adjChunks array.

  • removeOriginalChunks – [in] If true, original chunks that are merged are removed.
