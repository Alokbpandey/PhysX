NvBlastExtAssetUtilsAddExternalBonds¶

Defined in extensions/assetutils/NvBlastExtAssetUtils.h

NvBlastAsset *NvBlastExtAssetUtilsAddExternalBonds(const NvBlastAsset *asset, const uint32_t *externalBoundChunks, uint32_t externalBoundChunkCount, const NvcVec3 *bondDirections, const uint32_t *bondUserData)¶

Reauthor the provided asset to create external bonds in the specified support chunks.

Parameters

  • asset – [in] Pointer to the original asset. Won’t be modified.

  • externalBoundChunks – [in] Array of support chunk indices which are to be bound to the external body.

  • externalBoundChunksCount – [in] Size of externalBoundChunks array.

  • bondDirections – [in] Array of normals for each bond (size externalBoundChunksCount)

  • bondUserData – [in] Array of user data values for the new bonds, of size externalBoundChunksCount. May be NULL. If NULL, bond user data will be set to zero.

Returns

a new asset with added bonds if successful, NULL otherwise.
