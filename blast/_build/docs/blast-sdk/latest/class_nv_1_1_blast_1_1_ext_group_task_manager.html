Nv::Blast::ExtGroupTaskManager¶

Defined in extensions/physx/NvBlastExtPxTask.h

class Nv::Blast::ExtGroupTaskManager¶

Uses a physx::PxTaskManager to process a TkGroup concurrently.

Public Functions

virtual void setGroup(TkGroup*) = 0¶

Set the group to process.

Cannot be changed while a group being processed.

virtual uint32_t process(uint32_t workerCount = 0) = 0¶

Start processing the group.

The parallelizing strategy is to have all worker tasks running concurrently. The number of started tasks may be smaller than the requested value, when the task manager’s dispatcher thread count or the number of group jobs are smaller.

Parameters

workerCount – [in] The number of worker tasks to start, 0 uses the dispatcher’s worker thread count.

Returns

The number of worker tasks started. If 0, processing did not start and wait() will never return true.

virtual bool wait(bool block = true) = 0¶

Wait for the group to end processing.

When processing has finished, TkGroup::endProcess is executed.

Parameters

block – [in] true: does not return until the group has been processed. false: return immediately if workers are still processing the group.

Returns

true if group processing was completed (and the group was actually processing)
