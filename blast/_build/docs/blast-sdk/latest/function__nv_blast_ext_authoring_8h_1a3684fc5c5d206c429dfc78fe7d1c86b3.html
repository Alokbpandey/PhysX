NvBlastExtAuthoringTransformCollisionHullInPlace¶

Defined in extensions/authoring/NvBlastExtAuthoring.h

void NvBlastExtAuthoringTransformCollisionHullInPlace(Nv::Blast::CollisionHull *hull, const NvcVec3 *scaling, const NvcQuat *rotation, const NvcVec3 *translation)¶

Transforms collision hull in place using scale, rotation, transform.

Parameters

  • hull – [inout] Pointer to the hull to be transformed (modified).

  • scale – [in] Pointer to scale to be applied. Can be nullptr.

  • rotation – [in] Pointer to rotation to be applied. Can be nullptr.

  • translation – [in] Pointer to translation to be applied. Can be nullptr.
