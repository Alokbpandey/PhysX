class Nv::Blast::TkIdentifiable : public Nv::Blast::TkObject¶

TkIdentifiable objects have getID and setID methods for individual objects.

They also have a type (class) identification.

Subclassed by Nv::Blast::TkAsset, Nv::Blast::TkFamily, Nv::Blast::TkGroup
