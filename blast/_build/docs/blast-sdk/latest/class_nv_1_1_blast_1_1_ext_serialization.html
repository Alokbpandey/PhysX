class ExtSerialization

Serialization manager interface.

Public Functions

virtual bool setSerializationEncoding(uint32_t encodingID) = 0

Set the serialization encoding to use.

(See EncodingID.)

Returns

true iff successful.

virtual uint32_t getSerializationEncoding() const = 0

Retrieve the current serialization encoding being used.

Note, by default this is set to the encoding of the first serializer registered by a module. Currently this is done automatically by the NvBlastExtLlExtension module.

Returns

the current serialization encoding (zero if none is set).

virtual void setBufferProvider(BufferProvider *bufferProvider) = 0

Set the buffer provider callback to use.

(See BufferProvider.) If not set, a default provider using NVBLAST_ALLOC (see NvBlastGlobals.h) is used, which may be freed using NvBLAST_FREE.

Parameters

bufferProvider – [in] Buffer provider callback to use. If NULL, uses the default provider using the allocator given in NvBlastGlobals.

virtual bool peekHeader(uint32_t *objectTypeID, uint32_t *encodingID, uint64_t *dataSize, const void *buffer, uint64_t bufferSize) = 0

Reads information from a buffer, returning the contained object type ID, encoding ID, and data size.

Parameters

  • objectTypeID – [out] If not NULL, the object type ID is written to *objectTypeID.

  • encodingID – [out] If not NULL, the encoding ID is written to *encodingID.

  • dataSize – [out] If not NULL, the data size is written to *dataSize. (Does not include the size of the header.)

  • buffer – [in] Pointer to the buffer to read.

  • bufferSize – [in] Size of the buffer to read.

Returns

true iff the header is successfully read.

virtual const void *skipObject(uint64_t &bufferSize, const void *buffer) = 0

Determines the current object in the buffer and returns the position in the buffer immediately after the object.

Parameters

  • bufferSize – [inout] Size of the buffer to read on input, on output the remaining buffer size given the return buffer value.

  • buffer – [in] Pointer to the buffer to read.

Returns

a pointer to the new position in the buffer after the skipped object if successful, NULL otherwise. The bufferSize field is only updated if a valid pointer is returned.

virtual void *deserializeFromBuffer(const void *buffer, uint64_t bufferSize, uint32_t *objectTypeIDPtr = nullptr) = 0

Deserialize from a buffer into a newly allocated object.

Parameters

  • buffer – [in] Pointer to the buffer to read.

  • bufferSize – [in] Size of the buffer to read.

  • objectTypeIDPtr – [out] Optional, if not NULL then *objectTypeIDPtr will be filled with the deserialized object’s type ID if deserialization is successful, or 0 if unsuccessful.

Returns

object pointer; returns null if failed to deserialize.

virtual uint64_t serializeIntoBuffer(void *&buffer, const void *object, uint32_t objectTypeID) = 0

Serialize into a buffer.

Allocates the buffer internally using the callack set in setBufferProvider.

Parameters

  • buffer – [out] Pointer to the buffer created.

  • object – [in] Object pointer.

  • objectTypeID – [in] Object type ID.

Returns

the number of bytes serialized into the buffer (zero if unsuccessful).
