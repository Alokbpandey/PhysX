extensions/authoring/NvBlastExtAuthoring.h¶

↰ Parent directory: extensions/authoring/

Defines the API for the NvBlastExtAuthoring blast sdk extension.

Source: extensions/authoring/NvBlastExtAuthoring.h

Namespaces¶

  • Nv: FractureTool has requirements to input meshes to fracture them successfully: 1) Mesh should be closed (watertight) 2) There should not be self-intersections and open-edges.

  • Nv::Blast

Functions¶

  • NvBlastExtAuthoringBuildCollisionMeshes: Build collision meshes.

  • NvBlastExtAuthoringBuildCutoutSet: Builds a cutout set (which must have been initially created by createCutoutSet()).

  • NvBlastExtAuthoringBuildMeshConvexDecomposition: Build convex mesh decomposition.

  • NvBlastExtAuthoringCreateBBoxBasedAccelerator: Create BBoxBasedAccelerator - SpatialAccelerator which uses a bbox/grid algorithm.

  • NvBlastExtAuthoringCreateBondGenerator: Create BlastBondGenerator.

  • NvBlastExtAuthoringCreateBooleanTool: Create BooleanTool object.

  • NvBlastExtAuthoringCreateCutoutSet: Instantiates a blank CutoutSet.

  • NvBlastExtAuthoringCreateFractureTool: Create FractureTool object.

  • NvBlastExtAuthoringCreateGridAccelerator: Create GridAccelerator - SpatialAccelerator which use Grid for faster mesh sampling.

  • NvBlastExtAuthoringCreateMesh: Constructs mesh object from array of triangles.

  • NvBlastExtAuthoringCreateMeshCleaner: Creates MeshCleaner object.

  • NvBlastExtAuthoringCreateMeshFromFacets: Constructs mesh object from array of vertices, edges and facets.

  • NvBlastExtAuthoringCreateMeshOnlyTriangles: Constructs mesh object from triangles represented as arrays of vertices, indices and per facet material.

  • NvBlastExtAuthoringCreatePatternGenerator: Returns pattern generator used for generating fracture patterns.

  • NvBlastExtAuthoringCreateSpatialGrid: Create spatial grid for mesh.

  • NvBlastExtAuthoringCreateSweepingAccelerator: Create SweepingAccelerator - SpatialAccelerator which uses a sweep algorithm.

  • NvBlastExtAuthoringCreateVoronoiSitesGenerator: Voronoi sites should not be generated outside of the fractured mesh, so VoronoiSitesGenerator should be supplied with fracture mesh.

  • NvBlastExtAuthoringFindAssetConnectingBonds: Finds bonds connecting chunks in a list of assets.

  • NvBlastExtAuthoringProcessFracture: Performs pending fractures and generates fractured asset, render and collision geometry.

  • NvBlastExtAuthoringReleaseAuthoringResult: Releases AuthoringResult data.

  • NvBlastExtAuthoringReleaseAuthoringResultCollision: Releases collision data for AuthoringResult.

  • NvBlastExtAuthoringTransformCollisionHull: Transforms collision hull in place using scale, rotation, transform.

  • NvBlastExtAuthoringTransformCollisionHullInPlace: Transforms collision hull in place using scale, rotation, transform.

  • NvBlastExtAuthoringTrimCollisionGeometry: Convex geometry trimming.

  • NvBlastExtAuthoringUpdateGraphicsMesh: Updates graphics mesh only.
