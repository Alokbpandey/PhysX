lowlevel/NvBlast.h

↰ Parent directory: lowlevel/

Defines the API for the low-level blast library.

Source: lowlevel/NvBlast.h

Functions

  • NvBlastActorApplyFracture: Applies the direct fracture and breaks graph bonds/edges as necessary.

  • NvBlastActorCacheBondHeath: Tell the system to cache the bond health for the given bond index.

  • NvBlastActorCanFracture: Determines if the actor can fracture further.

  • NvBlastActorDeactivate: Deactivate an actor within its family.

  • NvBlastActorGenerateFracture: Creates fracture commands for the actor using a damage program and program parameters (material and damage descriptions).

  • NvBlastActorGetBondHealths: Access the bond health data for an actor.

  • NvBlastActorGetCachedBondHeaths: Access the cached bond health data for an actor.

  • NvBlastActorGetFamily: Access to an actor’s family.

  • NvBlastActorGetGraphNodeCount: Get the number of graph nodes for this actor.

  • NvBlastActorGetGraphNodeIndices: Retrieve a list of graph node indices for the actor into the given array.

  • NvBlastActorGetIndex: Access to an actor’s internal index.

  • NvBlastActorGetMaxActorCountForSplit: Returns the upper-bound number of actors which can be created by calling NvBlastActorSplit with that actor, this value can’t exceed chunk count.

  • NvBlastActorGetRequiredScratchForSplit: Returns the number of bytes of scratch memory that the user must supply to NvBlastActorSplit, based upon the actor that will be passed into that function.

  • NvBlastActorGetSerializationSize: The buffer size needed to serialize a single actor.

  • NvBlastActorGetVisibleChunkCount: Get the number of visible chunks for this actor.

  • NvBlastActorGetVisibleChunkIndices: Retrieve a list of visible chunk indices for the actor into the given array.

  • NvBlastActorHasExternalBonds

  • NvBlastActorIsSplitRequired: Determines if the actor is damaged (was fractured) and split call is required.

  • NvBlastActorSerialize: Serialize a single actor to a buffer.

  • NvBlastActorSplit: Releases the oldActor and creates its children newActors if necessary.

  • NvBlastApplyAssetDescChunkReorderMap: Apply chunk reorder map.

  • NvBlastApplyAssetDescChunkReorderMapInPlace: Apply chunk reorder map.

  • NvBlastAssetCreateFamily: Family-building function.

  • NvBlastAssetCreateFamilyFromSizeData: Family-building function.

  • NvBlastAssetGetActorSerializationSizeUpperBound: A buffer size sufficient to serialize an actor instanced from a given asset.

  • NvBlastAssetGetBondCount: Get the number of bonds in the given asset.

  • NvBlastAssetGetBonds: Access an array of bonds of the given asset.

  • NvBlastAssetGetChunkCount: Get the number of chunks in the given asset.

  • NvBlastAssetGetChunks: Access an array of chunks of the given asset.

  • NvBlastAssetGetChunkToGraphNodeMap: Access a map from chunk index to graph node index.

  • NvBlastAssetGetFamilyMemorySize: Calculates the memory requirements for a family based upon an asset.

  • NvBlastAssetGetFamilyMemorySizeFromSizeData: Calculates the memory requirements for a family based upon supplied sized data.

  • NvBlastAssetGetFirstSubsupportChunkIndex: Get the first subsupport chunk index in the given asset.

  • NvBlastAssetGetFormatVersion: Retrieve the data format version for the given asset.

  • NvBlastAssetGetID: Retrieve the asset ID.

  • NvBlastAssetGetLeafChunkCount: Get the number of leaf chunks in the given asset.

  • NvBlastAssetGetSize: Retrieve the memory size (in bytes) of the given data asset.

  • NvBlastAssetGetSupportChunkCount: Get the number of support chunks in the given asset.

  • NvBlastAssetGetSupportGraph: Access the support graph for the given asset.

  • NvBlastAssetMemSizeDataFromAsset: Fill out the size data from the provided asset.

  • NvBlastAssetSetID: Set an asset’s ID.

  • NvBlastBuildAssetDescChunkReorderMap: Build chunk reorder map.

  • NvBlastCreateAsset: Asset-building function.

  • NvBlastEnsureAssetExactSupportCoverage: Function to ensure (check and update) support coverage of chunks.

  • NvBlastFamilyCreateFirstActor: Instance the family’s asset into a new, unfractured actor.

  • NvBlastFamilyDeserializeActor: Deserialize a single Actor from a buffer into the given family.

  • NvBlastFamilyGetActorByIndex: Retrieve the actor associated with the given actor index.

  • NvBlastFamilyGetActorCount: Retrieve the number of active actors associated with the given family.

  • NvBlastFamilyGetActors: Retrieve the active actors associated with the given family.

  • NvBlastFamilyGetAsset: Retrieve the asset of the given family.

  • NvBlastFamilyGetAssetID: Retrieve the asset ID of the given family.

  • NvBlastFamilyGetChunkActor: Retrieve the actor associated with the given chunk.

  • NvBlastFamilyGetChunkActorIndices: Retrieve the actor indices associated with chunks.

  • NvBlastFamilyGetFormatVersion: Retrieve the data format version for the given family.

  • NvBlastFamilyGetMaxActorCount: Retrieve the max active actor count family could have.

  • NvBlastFamilyGetRequiredScratchForCreateFirstActor: Returns the number of bytes of scratch memory that the user must supply to NvBlastFamilyCreateFirstActor.

  • NvBlastFamilyGetSize: Retrieve the size (in bytes) of the given family.

  • NvBlastFamilySetAsset: Set asset to the family.

  • NvBlastGetAssetMemorySize: Calculates the memory requirements for an asset based upon its descriptor.

  • NvBlastGetAssetMemorySizeFromSizeData: Calculates the memory requirements for an asset based upon supplied sized data.

  • NvBlastGetRequiredScratchForCreateAsset: Returns the number of bytes of scratch memory that the user must supply to NvBlastCreateAsset, based upon the descriptor that will be passed into that function.

  • NvBlastReorderAssetDescChunks: Build and apply chunk reorder map.

  • NvBlastTicksToSeconds: Convert a tick value from NvBlastTimers to seconds.

  • NvBlastTimersReset: Resets all values in the given NvBlastTimers struct to zero.
