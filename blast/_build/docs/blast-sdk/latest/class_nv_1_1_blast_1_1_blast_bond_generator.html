virtual int32_t buildDescFromInternalFracture(FractureTool *tool, const bool *chunkIsSupport, NvBlastBondDesc *&resultBondDescs, NvBlastChunkDesc *&resultChunkDescriptors) = 0¶

This method based on marking triangles during fracture process, so can be used only with internally fractured meshes.

Note

User should call NVBLAST_FREE for resultBondDescs when it not needed anymore

Parameters

  • tool – [in] FractureTool which contains chunks representation, tool->finalizeFracturing() should be called before.

  • chunkIsSupport – [in] Pointer to array of flags, if true - chunk is support. Array size should be equal to chunk count in tool.

  • resultBondDescs – [out] Pointer to array of created bond descriptors.

  • resultChunkDescriptors – [out] Pointer to array of created chunk descriptors.

Returns

Number of created bonds

Note

User should call NVBLAST_FREE for resultBondDescs when it not needed anymore

Parameters

  • meshCount – [in] Number of meshes

  • geometryOffset – [in] Pointer to array of triangle offsets for each mesh. Containts meshCount + 1 element, last one is total number of triangles in geometry

  • geometry – [in] Pointer to array of triangles. Triangles from geometryOffset[i] to geometryOffset[i+1] correspond to i-th mesh.

  • overlapsCount – [in] Number of overlaps

  • overlaps – [in] Pointer to array of pairs - indexes of chunks, for which bond should be created.

  • resultBond – [out] Pointer to array of result bonds.

  • cfg – [in] Bond creation mode.

virtual int32_t bondsFromPrefractured(uint32_t meshCount, const uint32_t *convexHullOffset, const CollisionHull **chunkHulls, const bool *chunkIsSupport, const uint32_t *meshGroups, NvBlastBondDesc *&resultBondDescs, float maxSeparation) = 0¶

Creates bond description for prefractured meshes, when there is no info about which chunks should be connected with bond.

This uses the same process as bondsFromPrefractured using the BondGenMode::AVERAGE mode however the existing collision data is used.

Note

User should call NVBLAST_FREE for resultBondDescs when it not needed anymore.

Parameters

  • meshCount – [in] Number of meshes

  • convexHullOffset – [in] Pointer to array of convex hull offsets for each mesh. Containts meshCount + 1 element, last one is total number of hulls in the geometry

  • chunkHulls – [in] Pointer to array of convex hulls. Hulls from convexHullOffset[i] to convexHullOffset[i+1] correspond to i-th mesh.

  • chunkIsSupport – [in] Pointer to array of flags, if true - chunk is support. Array size should be equal to chunk count in tool.

  • meshGroups – [in] Pointer to array of group ids for each mesh, bonds will not be generated between meshs of the same group. If null each mesh is assumed to be in it’s own group.

  • resultBondDescs – [out] Pointer to array of result bonds.

Returns

Number of created bonds
