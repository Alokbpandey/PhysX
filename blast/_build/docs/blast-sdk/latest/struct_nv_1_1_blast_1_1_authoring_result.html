struct AuthoringResult

Authoring results.

Which contains NvBlastAsset, render and collision meshes. If it was created by NvBlast it should be released with NvBlastExtAuthoringReleaseAuthoringResult For releasing just collsion geometry call NvBlastExtAuthoringReleaseAuthoringResultCollision
