class Nv::Blast::TkEventListener¶

Interface for a listener of TkEvent data.

The user must implement this interface and pass it to the object which will dispatch the events.

Subclassed by Nv::Blast::ExtSync

virtual void receive(const TkEvent *events, uint32_t eventCount) = 0¶

Interface to be implemented by the user.

Events will be sent by BlastTk through a call to this function.

Parameters

  • events – [in] The array of events being dispatched.

  • eventCount – [in] The number of events in the array.
