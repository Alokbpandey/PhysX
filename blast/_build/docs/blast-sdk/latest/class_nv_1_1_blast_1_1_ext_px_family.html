class Nv::Blast::ExtPxFamily¶

PxFamily.

A collection of actors. Maps 1 to 1 with TkFamily.

Public Functions

virtual bool spawn(const physx::PxTransform &pose, const physx::PxVec3 &scale, const ExtPxSpawnSettings &settings) = 0¶

Spawn ExtPxFamily.

Can be called only once. Actual PhysX actors will created and placed in PxScene

Parameters

  • pose – [in] World transform.

  • scale – [in] Scale applied to spawned actors.

  • settings – [in] Spawn settings.

Returns

true if spawn was successful, false otherwise.

virtual bool despawn() = 0¶

Despawn this ExtPxFamily.

This removes the PhysX actors from PxScene and deletes them, as well as deleting the created ExtPxActors

This does not call release() on the family.

Returns

true if successful.
