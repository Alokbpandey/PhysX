class Nv::Blast::IJsonCollisionExporter¶

Interface to object which serializes collision geometry to JSON format.

Public Functions

virtual void release() = 0¶

Delete this object.

virtual bool writeCollision(const char *path, uint32_t chunkCount, const uint32_t *hullOffsets, const CollisionHull *const *hulls) = 0¶

Method creates file with given path and serializes given array of arrays of convex hulls to it in JSON format.

Parameters

  • path – [in] Output file path.

  • chunkCount – [in] The number of chunks, may be less than the number of collision hulls.

  • hullOffsets – [in] Collision hull offsets. Contains chunkCount + 1 element. First collision hull for i-th chunk: hull[hullOffsets[i]]. hullOffsets[chunkCount+1] is total number of hulls.

  • hulls – [in] Array of pointers to convex hull descriptors, contiguously grouped for chunk[0], chunk[1], etc.
