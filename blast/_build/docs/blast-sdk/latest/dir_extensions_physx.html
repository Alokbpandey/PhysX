extensions/physx/¶

↰ Parent directory: extensions/

Files¶

  • extensions/physx/NvBlastExtPx.h: Defines the API for the NvBlastExtPx blast sdk extension.

  • extensions/physx/NvBlastExtPxActor.h: Defines the API for the NvBlastExtPxActor class.

  • extensions/physx/NvBlastExtPxAsset.h: Defines the API for the NvBlastExtPxAsset class.

  • extensions/physx/NvBlastExtPxCollisionBuilder.h: Defines the API for the NvBlastExtPxCollisionBuilder, which extends ConvexMeshBuilder handle Px (physx)-specific blast data.

  • extensions/physx/NvBlastExtPxFamily.h: Defines the API for the NvBlastExtPxFamily class.

  • extensions/physx/NvBlastExtPxImpactDamageManager.h: Defines the API for the NvBlastExtPxDamageManager class, which processes impact damage from physx.

  • extensions/physx/NvBlastExtPxListener.h: Defines the API for the NvBlastExtPxListener class, which provides actor created/destroyed event callbacks.

  • extensions/physx/NvBlastExtPxManager.h: Defines the API for the NvBlastExtPxManager class, a general container for all things ExtPx.

  • extensions/physx/NvBlastExtPxStressSolver.h: Defines the API for the ExtPxStressSolver, which wraps the basic stress solver (NvBlastExtStress) for use with physx.

  • extensions/physx/NvBlastExtPxSync.h: Defines the API for synchronization of damage and fracture events between two clients via an intermediate buffer.

  • extensions/physx/NvBlastExtPxTask.h: Defines a task manager API for multithreading ExtPx oerations.

  • extensions/physx/NvBlastPxCallbacks.h: Defines allocator and error callbacks for ExtPx.
