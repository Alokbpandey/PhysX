NvBlastExtAuthoringProcessFracture¶

Defined in extensions/authoring/NvBlastExtAuthoring.h

Nv::Blast::AuthoringResult *NvBlastExtAuthoringProcessFracture(Nv::Blast::FractureTool &fTool, Nv::Blast::BlastBondGenerator &bondGenerator, Nv::Blast::ConvexMeshBuilder &collisionBuilder, const Nv::Blast::ConvexDecompositionParams &collisionParam, int32_t defaultSupportDepth = -1)¶

Performs pending fractures and generates fractured asset, render and collision geometry.

Parameters

  • fTool – [in] Fracture tool created by NvBlastExtAuthoringCreateFractureTool

  • bondGenerator – [in] Bond generator created by NvBlastExtAuthoringCreateBondGenerator

  • collisionBuilder – [in] Collision builder created by NvBlastExtAuthoringCreateConvexMeshBuilder

  • defaultSupportDepth – [in] All new chunks will be marked as support if its depth equal to defaultSupportDepth. By default leaves (chunks without children) marked as support.

  • collisionParam – [in] Parameters of collision hulls generation.

Returns

Authoring result
