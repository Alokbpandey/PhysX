static ExtPxManager *create(physx::PxPhysics &physics, TkFramework &framework, ExtPxCreateJointFunction createFn = nullptr, bool useUserData = true)¶

Create a new ExtPxManager.

Parameters

  • physics – [in] The PxPhysics instance to be used by ExtPxManager.

  • framework – [in] The TkFramework instance to be used by ExtPxManager.

  • cooking – [in] The optional PxCooking. Required for collision builder.

  • createFn – [in] The function to be used when creating joints, can be nullptr.

  • useUserData – [in] Flag if ExtPxManager is allowed to override PxActor’s userData, it will store pointer to PxActor there. It is recommended as fastest way. If set to ‘false’ HashMap will be used.

Returns

the new ExtPxManager if successful, NULL otherwise.
